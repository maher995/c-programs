# C++ Basics Programs

This repository contains a collection of simple C++ programs that I wrote while completing a **C++ Basics** course. Each program focuses on a specific programming concept or problem to help build a strong foundation in C++.

## Topics Covered

- Variables and Data Types
- Input and Output
- Arithmetic Operations
- Functions & Procedures
- Mathematical Calculations
- Geometry Formulas
- Problem Solving
- Basic Programming Logic
- Loops
- continue & break statements
  
## Requirements

- C++ Compiler (G++, Clang, or MSVC)
- C++11 or later


## Purpose

The purpose of this repository is to practice the fundamentals of C++ programming through small, focused exercises. These programs were completed as part of a C++ Basics course and serve as reference implementations for beginner-level concepts.
